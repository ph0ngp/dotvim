## Requirements
- Latest neovim with required options:
  - On OS X:
  ```bash
  pip install neovim
  npm install -g neovim
  gem install neovim
  brew install neovim
  ```

- [Vim Plug](https://github.com/junegunn/vim-plug)

## Install
```bash
git clone https://github.com/phphong/dotvim ~/.config/nvim
nvim
```

Then, in Neovim, execute `:PlugInstall`
